## 🚀 Overview

**AI Platform** is a custom AI service layer built with JavaScript that lets applications access AI-powered features through authenticated API keys.  
It acts as a **gateway between client apps and AI engines**, allowing secure, manageable, and reusable AI access locally (Ollama) or via third-party providers.

The platform issues **custom API keys** and routes AI requests through components like:

✔ Local AI models using **Ollama**  
✔ Third-party AI services (e.g., Groq, OpenAI, Gemini)

---

## ✨ Features

- **🔑 API Key Management**  
  Generate, validate, and manage API keys for secure access.

- **🧠 AI Gateway Logic**  
  Acts as an intermediary between client requests and AI engines.

- **🤖 Local AI Support**  
  Works with **Ollama** models for offline, private AI inference.

- **☁️ External AI Integration**  
  Support for external AI services with API keys.

- **📡 Unified AI Endpoint**  
  Clients can call one API endpoint regardless of the underlying model.

- **⚙️ Extensible & Modular**  
  Easily add support for additional AI services or features.
