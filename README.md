# 🚀 AI Platform

Custom AI API Gateway with API Key Management.

## 🔥 Features
- API key authentication
- Unified AI API endpoint
- Supports Ollama (local AI)
- Supports external APIs (OpenAI, Gemini, etc.)
- Modular and scalable backend

## 🧠 Use Case
This platform allows developers to connect multiple AI providers through a single API endpoint with secure access control.

## 📡 Example API Usage

POST /api/ai/generate

Headers:
Authorization: Bearer YOUR_API_KEY

Body:
{
  "prompt": "Explain MERN stack"
}

- Node.js
